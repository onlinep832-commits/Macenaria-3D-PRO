# Planejador 3D - Pro

## Descrição
Projeto web para planejamento 3D de ambientes.


## Como rodar
1. Instale as dependências necessárias (Node.js e npm).
2. Execute `npm install` para instalar as dependências.
3. Execute `node server.js` na raiz do projeto para iniciar o backend e o servidor de arquivos estáticos.
4. Acesse http://localhost:3000/public/login.html ou http://localhost:3000/public/register.html no navegador.

> Não utilize o live-server, pois ele não repassa as rotas de API e causará erros de conexão.

## Estrutura de pastas
- public/: arquivos finais para produção
- src/: código fonte (opcional)
- docs/: documentação
- tests/: testes automatizados
- assets/: imagens, ícones, fontes
